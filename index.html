Facilidade de Uso:
O grid personalizado exige mais conhecimento técnico e tempo para ser implementado, enquanto o Bootstrap oferece um sistema pronto e fácil de usar.
no final tive muita facilidade em usar
Flexibilidade:
O grid personalizado oferece controle total, permitindo ajustes específicos, enquanto o Bootstrap é flexível dentro de seus limites, adaptando-se bem a casos comuns.
Personalização:
No grid personalizado, tudo pode ser ajustado conforme necessário. O Bootstrap permite personalização, mas dentro de suas convenções, sendo mais difícil fazer grandes mudanças sem sobrescrever muito código.
